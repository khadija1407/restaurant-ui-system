# 🍽️ Online Restaurant System Frontend

A modern and responsive **Online Restaurant System Frontend** built under time constraints by decoding a problem statement through a game-based activity.

## 🚀 Project Overview
This project was developed as part of a club activity where the challenge was to **identify the problem statement through a game** and build a working solution within **30 minutes**.

It focuses on creating a smooth and user-friendly restaurant experience through an intuitive UI.

## ✨ Features
- 🍔 Interactive Menu Browsing  
- 🪑 Online Table Booking Interface  
- 🛒 Food Ordering UI  
- 📱 Fully Responsive Design  
- ⚡ Clean and Modern UI  

## 🛠️ Tech Stack
- React  
- Tailwind CSS  
- Vite  

## 🎯 Learning Outcomes
- ⏱️ Rapid problem-solving under time constraints  
- 🧠 Converting abstract ideas into functional UI  
- ⚡ Fast development and decision making  
- 🎨 Improving UI/UX skills
